api2.orozcoh.com:3000

Arbol de rutas de esta API dedicada a guardar
Urls varias
-------------------------------------------------------------------------------
PUBLIC:
 
  - api2.orozcoh.com:3000/v1/urls 
      - GET: Last 60 logs

  - api2.orozcoh.com:3000/v1/urls/{id} 
      - GET: Get the data of the ID given

  - api2.orozcoh.com:3000/v1/urls/getAll 
      - GET: All urls POSTed

  - api2.orozcoh.com:3000/v1/urls/getLast 
      - GET: Last url POSTed

  - api2.orozcoh.com:3000/v1/urls/lastUpdated 
      - GET: Time of last update
--------------------------------------------------------------------------------

ALLOWED USERS:
 
  - api2.orozcoh.com:3000/v1/urls
 
      - POST: Only for allowed devices with API_KEY 
  - api2.orozcoh.com:3000/v1/urls/{timestamp}
 
      - PATCH: Update info (in this data logger, this should not be possible)
      - DELETE: Delete selected data (in this data logger, this should not be possible)
